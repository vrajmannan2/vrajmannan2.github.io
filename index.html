How to create a Github Webpage

First you would like to go to GitHub and make a new account if you already do not have one.
Then, you would like to go to create new repository name it whatever you like then create the repository.
After that, you would like to click upload files and make a index.html file with whatever you like and upload it.
At last, you click commit and then you can go onto your webpage!
